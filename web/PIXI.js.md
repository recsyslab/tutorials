# PIXI.js

## 開発環境の構築

### プロジェクトのホームディレクトリの作成
```bash
$ cd ~/dev/
$ mkdir -p ~/dev/pixi/frontend/
$ mkdir -p ~/dev/pixi/backend/
```

## node.jsのバージョンの確認
```bash
$ node --version
```

## node.jsパッケージのセットアップ
```bash
$ cd ~/dev/pixi/frontend/
$ npm init
$ ls
$ less package.json
```

## TypeScript関連モジュールのインストール
```bash
$ npm i --save-dev typescript tslint tslint-config-airbnb typedoc
$ less package.json
```
