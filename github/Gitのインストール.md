# Gitのインストール

## インストール
```bash
sudo apt install git
```

## Gitのバージョンの確認
```bash
git --version
```

## ユーザ名とメールアドレスの登録
```bash
git config --global user.name "【ユーザ名】"
git config --global user.email "【メールアドレス】"
```

## color.uiの設定
```bash
git config --global color.ui auto
```

## 設定ファイルの確認
```bash
cat ~/.gitconfig
```
