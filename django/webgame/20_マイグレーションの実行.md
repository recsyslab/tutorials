# マイグレーションの実行

これで，データベースの設定ができました．`manage.py`があるディレクトリにいることを確認したうえで，以下のコマンドを実行してください．ただし，【ユーザ名】と【パスワード】には，それぞれデータベースに接続するユーザ名とパスワードを入力してください．

```bash
$ export DB_USER=【ユーザ名】
$ export DB_PASSWORD=【パスワード】
$ python manage.py migrate
```
