# システム情報等の確認

## ディストリビューションバージョンの確認
```bash
lsb_release -a
```

## アーキテクチャの確認
```bash
arch
```

## ホスト名の確認
```bash
hostname
```

## IPアドレスの確認
```bash
ifconfig
```

## スペック（ディスクサイズ，CPU，メモリ）の確認
```bash
df -h
cat /proc/cpuinfo | grep 'model name'
cat /proc/cpuinfo | grep processor
cat /proc/meminfo | grep MemTotal
```
